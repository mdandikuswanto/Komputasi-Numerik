    Metode Monte Carlo dan Simulasi¶

    Metode Monte Carlo adalah algoritma komputasi untuk mensimulasikan berbagai perilaku sistem fisika dan matematika. Penggunaan metode ini adalah untuk mengevaluasi integral definit, terutama integral multidimensi dengan syarat dan batasan yang rumit.

    Metode ini terbukti efisien dalam memecahkan persamaan diferensial integral medan radians, sehingga metode ini digunakan dalam perhitungan iluminasi global yang menghasilkan gambar-gambar fotorealistik model tiga dimensi, dimana diterapkan dalam video games, arsitektur, perancangan, film yang dihasilkan oleh komputer, efek-efek khusus dalam film, bisnis, ekonomi, dan bidang lainnya. Metode Monte Carlo pada umumnya dilakukan menggunakan komputer, dan memakai berbagai teknik simulasi komputer.

    Algoritme ini memerlukan pengulangan (repetisi) dan perhitungan yang amat kompleks, metode Monte Carlo pada umumnya dilakukan menggunakan komputer, dan memakai berbagai teknik simulasi komputer.

    Simulasi Monte Carlo adalah percobaan elemen probabilitas atau peluang dengan menggunakan sampel random atau acak untuk memodelkan sistem nyata.

    Langkah-langkah dalam Simulasi Monte Carlo :

    • Menentukan distribusi probabilitas untuk beberapa variabel penting.
    • Nembuat distribusi probabilitas kumulatif untuk setiap variabel pada langkah pertama.
    • Menentukan interval bilanga acak untuk setiap variabel pada langkah kedua.
    • Mebangkitkan bilangan acak.
    • Melakukan serangkaian simulasi percobaan.

    Angka Acak¶

    Angka acak adalah angka yang dipilih dari kumpulan angka terbatas atau tidak terbatas yang tidak memiliki pola prediksi untuk prediksi.

    Generator angka acak adalah perangkat yang dapat menghasilkan satu atau banyak angka acak dalam ruang lingkup yang ditentukan. Penghasil bilangan acak dapat berupa penghasil bilangan berbasis perangkat keras atau pseudo-acak.

    Di bawah ini adalah garis besar dari dua algoritma penghasil angka acak lainnya:

    1. Mother of All Pseudo-Random-Number Generators

    Inisialisasi empat nilai x_0, x_1, x_2, x_3 dan c ke nilai acak berdasarkan nilai seed. Membiarkan s = 2111111111 X_{n − 4} + 1492X_{n − 3} + 1776X_{n − 2} + 5115X_{n − 1} + c, hitung X_n = s mod (2^{32}) dan c = s / 2^{32} untuk n \geqslant 4. Diciptakan oleh GeorgeMarsaglia.

    1. rand() in Unix

    Inisialisasi x_0 ke nilai acak berdasarkan nilai seed. Hitung X_{n + 1} = (1103515245X_n +12345) mod (2^{31}) untuk n \geqslant 1.

    Tugas Programming :¶

    Penyelesaian :

    Soal nomor 3

    Tulis program untuk memverifikasi secara numerik bahwa \pi =\int_{0}^{2}(4-x^2)^{1/2}\, dx. Gunakan metode Monte Carlo dan 2500 angka acak.

    Listing Program :

    from scipy import random
import numpy as np
import matplotlib.pyplot as plt

b = 0
a = 2
n = 2500
def fungsi(x):
    return (4-x**2)**0.5

area = []
for i in range(n):
    xacak = np.zeros(n)

    for i in range(len(xacak)):
        xacak[i] = random.uniform(b,a)
        integral = 0.0

    for i in range(n):
        integral+=fungsi(xacak[i])

    jawab = (a-b)/float(n)*integral
    area.append(jawab)

plt.title("Hasil phi")
plt.hist(area,bins = 30, ec = 'black')
plt.xlabel("Area")
plt.show()

    Hasil Running (output) :

    Penjelasan :

    1. Inisialisasi sesuai soal, dimana variabel a merupakan integral batas atas yang nilainya = 2, variabel b merupakan integral batas bawah yang nilainya = 0, dan N adalah jumlah angka random yang akan diulang.

    2. Inisialisasikan rumus (4−x^2)^{\frac{1}{2}} dengan sebuah fungsi.

    3. Ulang i sebanyak n, yang nantinya akan membuat variabel baru bernama xacak untuk diisi angka desimal nantinya.

    4. dilanjut dengan looping di dalam looping lagi sebanyak jumlah angka yang ada di variabel xacak, setelah itu diisi dengan angka baru di antara 0 - 2 sesuai dengan batas bawah dan batas atas integral sehingga akan mendapatkan angka desimal baru. Pada baris selanjutnya angka tersebut dijumlahkan dengan inisialisasi integral yang bernilai 0.0 dengan angka desimal acak yang sudah dibuat sebelumnya dengan menggunakan rumus fungsi terlebih dahulu.

    5. Kemudian di hitung kembali ke dalam rumus {\frac{a-b}{n}\int} dimana selisih batas atas dan batas bawah dibagi dengan jumlah desimal random sebelumnya. Dan ditambahkan ke list variabel area dimana hasil tersebut nantinya digambarkan ke dalam diagram dengan fungsi matplotlib

    Soal nomor 4

    Gunakan metode Monte Carlo untuk memperkirakan integral

    \int_{-1}^{1}\int_{-1}^{1}\int_{-1}^{1}(x^2+y^2+z^2)dx\,dy\,dz

    Listing Program :

    from scipy import random
import numpy as np
b = -1
a = 1
n = 100
xacak=np.zeros(n)
yacak=np.zeros(n)
zacak=np.zeros(n)
integral=0.0
for i in range(4):
    for i in range(len(xacak)):
        xacak[i]=random.uniform(b,a)

    for i in range(len(yacak)):
        yacak[i]=random.uniform(b,a)

    for i in range(len(zacak)):
        zacak[i]=random.uniform(b,a)

    def fungsi(x,y,z):
        return (x**2)+(y**2)+(z**2)

    for i in range(n):
        integral+=fungsi(xacak[i],yacak[i],zacak[i])

hasil = (a-b)/float(n)*integral
print("hasil: ",jawab)

    Hasil Running (output) :

    hasil running hasil: 8.123944678902708

    Penjelasan :

    1. Inisialisasikan batas bawah, batas atas, dan jumlah angka.
    2. Inisialisasikan juga nilai x, y, dan z yang akan diisi angka random nantinya serta integral = 0.0 yang akan digunakan untuk jumlah akhir.
    3. Lakukan looping sebanyak 4 kali , dimana di dalam tersebut dilakukan perulangan sebanyak N untuk mengisi nilai random x, y, z serta perhitungan angka random tersebut dengan fungsi yang ada.
    4. Hasilnya dihitung menggunakan rumus {\frac{a-b}{n}\int}.
